# spotimy-music

> A web application for tracking todo lists.

### Demo video

## Technologies Used

-   React - version 3.5.1
-   Chakra UI - version 1.6.0
-   Node

## Setup

To run this project, install it locally using yarn or npm

yarn

```
$ cd ../todo/client
$ yarn
$ yarn start

$ cd ../todo/server
$ yarn
$ yarn start
```

npm

```
$ cd ../todo/client
$ npm install
$ npm start

$ cd ../todo/server
$ npm install
$ npm start
```
